# Printer Template Guide

## Purpose

Printer templates control how barcode stickers are produced for browser/PDF, ZPL, and EPL output.

## Configurable Settings

- label width
- label height
- DPI
- barcode position
- barcode height
- text position
- text size

Use the test-print feature with a sample barcode before printing a production batch.
