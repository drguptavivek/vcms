# Printer Templates User Guide

## Workflow

1. Open `Printer Templates`.
2. Choose output type: `Browser/PDF`, `ZPL`, or `EPL`.
3. Enter label width, height, DPI, barcode height, and X/Y positions.
4. Save the template.
5. Select the template when generating or reprinting barcode batches.
